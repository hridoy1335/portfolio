# 🚀 Portfolio System (Firebase + Admin Panel)

## Features
- Public portfolio website
- Admin panel to add projects
- Firebase Firestore database
- Ready for GitHub + Firebase Hosting

## Setup

### 1. Install Firebase CLI
npm install -g firebase-tools

### 2. Login
firebase login

### 3. Initialize project
firebase init

### 4. Add Firebase config
Replace firebaseConfig in:
- public/app.js
- admin/admin.js

### 5. Deploy
firebase deploy

## Admin Flow
/login → dashboard → add project → auto show in portfolio
